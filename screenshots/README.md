# Package screenshots

PNG files listed in `pubspec.yaml` appear on the pub.dev package page.

## Current

- `light.png` — example app on Android, light theme
- `dark.png` — example app on Android, dark theme

Source captures may be kept as `.jpg`; publish PNGs only (pub.dev requires PNG).

## README images

Use absolute GitHub raw URLs in `README.md` (relative paths do not render on
pub.dev), for example:

```markdown
![Light theme](https://raw.githubusercontent.com/balajimalathi/shadcn-chart/main/screenshots/light.png)
```

Recommended size: 1280x720 or similar 16:9.
