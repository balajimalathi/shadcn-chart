# shadcn_chart example

Demonstrates all seven chart types from the parent package.

## Run

```bash
cd example
flutter pub get
flutter run
```

## What to try

- Theme segmented control (system / light / dark)
- Refresh FAB to push new random data via widget rebuild
- Scroll the list to compare chart types

## Platform notes

This example uses `flutter_inappwebview`. Complete that package's platform setup
before running on iOS, Android, macOS, Windows, or web.
